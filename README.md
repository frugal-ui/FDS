# FDS
Development Support package

## Requirements
- esbuild: used to build the project
- frugalui
```shell
npm install esbuild @frugal-ui/base
```

## Install
`npm install @frugal-ui/fds`

## Usage
### Initialize
```shell
npm init
npx fds init
```

### Developing
- Entry point: `src/index.ts`
- Refer to the FrugalUI documentation

### Building
`npm run build`
